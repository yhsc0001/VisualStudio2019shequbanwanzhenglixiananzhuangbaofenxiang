## 安装步骤

1. **导入证书**：在安装之前，请先进入 `certificates` 目录，将目录中的三个证书文件全部导入到系统中。
2. **运行安装程序**：证书导入完成后，运行 `vs_setup.exe` 文件，按照提示完成安装过程。
